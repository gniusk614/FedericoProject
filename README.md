## 기술스택

### Client
* JavaScript ES5
* Jqeury 
* BootStrap5

### Server
* Java 1.8
* Spring 5.2.4


### DataBase
* oracle 12c
* mybatis 3.3.1


### deployment
* aws EC2 ubuntu
* aws rds  
<br/>

## 기능 상세
**본 웹사이트는 고객 페이지, 가맹점 페이지, 본사 페이지로 나뉘어있습니다.**
* 고객 페이지
  + 회원가입
  + 장바구니 및 주문, 주문조회
  + 매장찾기 (지역별, 지역명, 현위치기준)
  + 이벤트/공지사항 게시판
  + 고객문의
<br/>

* 가맹점 페이지
  + 고객주문 조회 및 처리
  + 자재발주 및 조회
  + 통계
  + 단골고객 등록 및 관리
<br/>

* 본사 페이지
  + 가맹점/본사계정 회원가입
  + 자재 등록 및 관리
  + 가맹점 자재발주 조회 및 처리
  + 메뉴 등록 및 관리
  + 이벤트/공지사항 게시판 관리
  + 고객문의 관리
  + 통계
